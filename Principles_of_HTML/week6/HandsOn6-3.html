Web Design
HTML HTML codes are special characters or character entities that can be used in web pages to display characters that otherwise would be difficult to type or use.
XML XML stands for eXtensible Markup Language. XML is designed to transport and store data. XML is important to know, and very easy to learn.
XHTML XHTML stands for EXtensible HyperText Markup Language XHTML is almost identical to HTML 4.01 XHTML is a stricter and cleaner version of HTML 4.01 XHTML is HTML defined as an XML application XHTML is supported by all major browsers.
