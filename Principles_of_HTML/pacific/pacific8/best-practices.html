Best Practices

The Pacific Trails Resort web site uses a general site map to organize the layout of the web site.
This is appropriate for the organization of the website because it is simple and the website content is very minimal.

3 Well Implemented Design Practices

  1. The web site displays without having to scroll horizontally.
  2. Navigation is easy to use for the target audinece.
  3. Common fonts are used.

3 Design Practices to Improve

  1. Should display on mobile device.
  2. Not as visually appealing as the site could be.
  3. Better contrast between background and text.

The Pacific Trails Resort web site could use many improvements. Such as the content needs to be improved as far as detail not so much in quantity.
I would try a simple slide show to show off more detail of the resort. The text color needs to be a darker color.
The font color is a little difficult for my young eyes to read.
